# Plant-Disease-Prediction-using-Deep-Learning-


1 . Downloaded PlantVillage dataset which contains 10 classes of Diseases which are specific to Tomato Plant . 

2 . Selected the Google collab Environment where all necessary libraries are imported .

3 . Developed the model 

4 . Saved the developed model as model.h5 in local storage 

5 . By using Flask , made web Interface where user can upload the random image which is related to the Tomato plant 
    and gets back the Output as Name of the disease through the prediction of the model
